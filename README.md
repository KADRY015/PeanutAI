PeanutAI adalah platform Agrotech berbasis web yang mengintegrasikan ilmu pertanian dengan teknologi Machine Learning. Sistem ini dirancang untuk mendeteksi penyakit foliar (daun) pada tanaman kacang tanah secara dini dan objektif melalui analisis citra digital.Sistem ini bekerja menggunakan metode Computer Vision untuk mengekstrak fitur warna rata-rata $R, G, B$ (Red, Green, Blue) dari foto daun yang diunggah, lalu mengklasifikasikannya menggunakan algoritma K-Nearest Neighbors (KNN) dengan parameter $K=1$ dan rumus jarak Euclidean Distance.Sistem ini bersifat transparan karena tidak hanya berfungsi sebagai sistem blackbox (langsung memunculkan hasil), tetapi juga menyediakan halaman simulator matematika untuk membuktikan kebenaran perhitungan manual algoritma secara teoretis sesuai dengan kaidah akademis.
 Panduan Penggunaan Sistem (User Guide)Aplikasi ini dirancang dengan antarmuka yang sangat mudah dipahami (User-Friendly). Berikut langkah-langkah penggunaannya berdasarkan menu navigasi:1. Membuka Dashboard Utama (🏠 Dashboard)Saat pertama kali web dibuka, pengguna akan disambut oleh halaman utama.Halaman ini menampilkan informasi ringkas platform, metrik akurasi sistem ($88.3\%$), jumlah dataset basis, serta urgensi mengapa sistem ini dibuat untuk membantu petani.2. Melakukan Deteksi Penyakit (🔍 Deteksi AI)Masuk ke menu Deteksi AI.Klik tombol "Browse files" atau seret (drag and drop) foto daun kacang tanah yang ingin diperiksa (format .jpg atau .png).Sistem akan memproses gambar secara otomatis (mengekstrak warna piksel).Output: Dalam hitungan detik, web akan menampilkan kartu hasil diagnosis (apakah daun Sehat, terkena Karat Daun, atau Bercak Daun) lengkap dengan nilai $RGB$ terekstrak serta Rekomendasi Tindakan Ahli untuk penanganan tanaman tersebut.3. Menguji Hitungan Teoretis (🧮 Simulator Manual)Menu ini digunakan khusus untuk kebutuhan akademis (pembuktian dokumen laporan).Pengguna dapat memasukkan angka piksel warna secara acak pada kolom Input Red, Green, dan Blue.Klik tombol "Jalankan Kalkulasi Jarak Euclidean".Output: Sistem akan menjabarkan rumus matematika dan hasil hitungan jarak dari angka inputan tadi ke semua data latih yang ada di database secara step-by-step hingga ditarik kesimpulan akhirnya.4. Membaca Pustaka Gejala (📚 Ensiklopedia)Pengguna dapat memilih tabs tanaman untuk membaca karakteristik morfologi gejala penyakit (Karat Daun dan Bercak Daun) secara teoretis berdasarkan literatur pakar agrikultur.5. Melihat Informasi Pengembang (👥 Tim Kami)Menampilkan profil kelompok mahasiswa (maksimal 4 orang) beserta pembagian role tugas masing-masing dalam menyelesaikan proyek Mid Test ini.
